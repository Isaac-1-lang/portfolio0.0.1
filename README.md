# Project

This project is built with Vite, TypeScript, React, shadcn-ui, and Tailwind CSS.

## Local Development

Requirements: Node.js and npm.

Steps:

```sh
git clone https://github.com/Isaac-1-lang/portfolio0.0.1
cd niylabs-interactive-hub
npm i
npm run dev
```

## Build

```sh
npm run build
```
